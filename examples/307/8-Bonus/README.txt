This folder contains some bonus files that may help some or other way.

It contains:
	UDL-Fift.xml: XML definition for Notepad++'s UDL feature for .fif files
	UDL-FunC.xml: XML definition for Notepad++'s UDL feature for .func files
	UDL-Sample.jpg - Example of how this UDLs would look in Notepad++
